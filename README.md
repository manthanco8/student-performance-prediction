# 🎓 Student Performance Prediction System

## 📌 Overview

This project is a Machine Learning-based system that predicts whether a student will pass or fail based on key academic factors.

---

## 🎯 Problem Statement

It is difficult for teachers to identify weak students early. This project helps in predicting student performance so that necessary actions can be taken in advance.

---

## 💡 Solution

A Logistic Regression model is used to classify students into:

* Pass
* Fail

based on:

* Study Hours
* Attendance
* Previous Marks

---

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

---

## ⚙️ How It Works

1. Data is stored in a CSV file
2. Data is loaded using pandas
3. Model is trained using Logistic Regression
4. Predictions are made for new input
5. Graph is generated for visualization

---
